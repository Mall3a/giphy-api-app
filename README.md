

# Gif Expert App

This is a simple React app that consumes the Giphy API.  
It allows you to search for GIFs by category and displays the results in a responsive grid.

## What is this repository for?

* **Quick summary:** A React app that fetches GIFs from the Giphy API and displays them.
* **Version:** 0.1.0
* **Purpose:** Practice using APIs with React and building a simple interactive UI.
* [Learn Markdown](https://bitbucket.org/tutorials/markdowndemo)

## Tech Stack

- **Frontend:** React 17, React DOM  
- **HTTP Requests:** Fetch API  
- **Testing:** React Testing Library, Jest  
- **Performance:** web-vitals  
- **Scripts:** react-scripts
